# LFSSR-ESAPE

This is the PyTorch implementation of “Explicit Spatial-Angular Pattern Embedding for Heterogeneous Imaging-Oriented Light Field Spatial Super-Resolution”.

# Usage
You can use the released code to train or test your own model.

For 8× SR
1. run './8x/train.py' to train your own model
2. run './8x/test.py' to test the trained model


For 4× SR
1. run './4x/train.py' to train your own model
2. run './4x/test.py' to test the trained model

The pre-trained model (8× and 4x) can be downloaded via Baidu Drive：https://pan.baidu.com/s/1cYrV5BvvjHijtYVr6CWNhQ (key: SAPE)
